Frequently Asked Questions

Is it really free?

Yes. No trial period. No credit card. No premium tier. Just free, forever. We built this because we wanted it to exist, not to make money.

Do I need to create an account?

Nope. Just open the page and start using it. No email, no password, no "confirm your account" email.

Where is my data stored?

On our servers if you use the cloud version. On your server if you self-host. All data is stored in a SQLite database‚Äîsimple and portable.

Can I use this with my team?

Currently, it's designed for individual use or teams that share access to a single board. Team collaboration features may come in the future.

Is my data private?

Yes. We don't track, analyze, or sell your data. If you self-host, your data never leaves your server. Simple as that.

Can I export my data?

Absolutely. It's stored in SQLite, so you can export it anytime. If you self-host, you have direct access to the database file.

What tech stack is it built with?

Backend: FastAPI + SQLAlchemy + SQLite. Frontend: Vanilla JavaScript + PicoCSS. Clean, simple, fast.

Can I contribute to the project?

Yes! It's open source (MIT License). Fork it, improve it, send PRs. We welcome contributions.

Will you add [feature]?

Maybe! Our philosophy is "less is more," but we're open to suggestions. Open an issue on GitHub and let's discuss.

Why did you build this?

We were frustrated with overcomplicated project management tools. So we built what we always wanted: simple, fast, and beautiful.
